## 🧠 Cognitive Architecture: The PMCR-O Loop
This system implements a law-governed, strictly sequential reasoning loop designed for deterministic enterprise execution. Unlike stochastic chatbots, this colony utilizes **Backward Flow Crystallization** to ensure that once a solution is found, it is never lost.

### The 4-Phase Cycle
1. **PLAN [P]:** Strategy generation & constraint pre-check.
2. **MAKE [M]:** Declarative execution & token resolution.
3. **CHECK [C]:** Mandatory conflict gates & scoring (Threshold: 0.85).
4. **REFLECT [R]:** Terminal disposition & EarnedConstraint authoring.
